# MotivationBasedAssets

The Motivation-based Assets are Asset developed from TU Graz for the RAGE Project (http://rageproject.eu/).

# Motivation Assessment Asset

This asset will be implemented as a client-side component. It is on the one hand concerned with supplying a motivation model (data structure containing motivation-related data) for a certain player. Therefore a local stored domain model is loaded. On the other hand, it updates the motivational state of a player based on motivation model and evidences gained from the game.

# Motivation Adaption Asset

This asset will be implemented as client-side component. It is used for deciding if an intervention influencing the player’s motivational state is needed; it supplies the interventions and an instance of this intervention.
